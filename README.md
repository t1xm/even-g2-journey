# Journey - Even G2 Milestone Tracker
**A minimalist journey tracker for Even Realities G2. Track new habits, health milestones, personal projects and more - and see your progress at a glance on the glasses HUD.** 💚

<img width="1011" height="971" alt="image" src="https://github.com/user-attachments/assets/07eff60b-3079-4a80-a214-2c2402f190a2" />

## ✨ Features
-   🎯 **Track Anything:** Suited for new habits, health milestones, personal projects and more
-   🕶️ **HUD Dashboard:** View precise time-spent stats (years, months, weeks, days, and hours)
-   🗂️ **Multiple Journeys:** Add new journeys, switch between them and Edit / Delete them
-   💾 **Local Storage:** Data persists safely on your phone
-   🌍 **Multi-Language:** Automatically adapts to English, German, or French    

## 🚀 Quick Start
-  **Prerequisites:** Node.js (v20+), `@evenrealities/evenhub-cli` (installed globally)
- **Install Dependencies:** `npm install`
- **Run Dev Server:** `npm run dev`
- **Launch on G2:** `npx evenhub qr --http --port 5173` (Scan QR code with Even App)
- **Local Simulator:** `evenhub-simulator http://localhost:5173`
